Edit WordPress comment's info: post id, parent comment id, user id, author IP, date and author user agent.


== Description ==

Edit comment's info:

* post id (by drop-down list)
* parent comment id (by drop-down list)
* user id
* author IP
* comment date (with seconds)
* author User Agent
